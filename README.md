# 🍕 Pizza Sales Analysis Dashboard

1. 📊 Project Overview

The **Pizza Sales Analysis** project analyzes pizza sales data to gain insights into business performance using **SQL, Power Query, and Power BI**.
This project focuses on transforming raw sales data into meaningful insights through data cleaning, querying, and interactive visual dashboards.

The dashboard helps understand sales trends, customer ordering patterns, and overall business performance.

---

2. 🎯 Objectives

* Analyze total revenue generated from pizza sales
* Identify the most popular pizza types and categories
* Understand sales trends by day and month
* Evaluate order quantity and average order value
* Provide data-driven insights for better business decisions

---

3. 🛠️ Tools & Technologies

* SQL – Data extraction and business queries
* Power Query – Data cleaning and transformation
* Power BI – Data visualization and interactive dashboard creation

---

4. 📈 Key Metrics Analyzed

* Total Revenue
* Total Orders
* Total Pizzas Sold
* Average Order Value
* Average Pizzas per Order

---

5 📊 Dashboard Insights

The Power BI dashboard provides insights such as:

* Daily and monthly sales trends
* Best-selling pizza categories
* Top 5 most ordered pizzas
* Sales distribution by pizza size
* Peak order days and times

---

6.  Project Workflow

1. Data Collection – Import raw pizza sales dataset
2. Data Cleaning – Transform and prepare data using Power Query
3. Data Analysis – Use SQL queries to calculate key business metrics
4. Visualization – Build an interactive Power BI dashboard

---

7. 🚀 Project Outcome

This project demonstrates how data analysis and visualization tools can transform raw sales data into actionable business insights.
